(PECL parle >= 0.7.0)

Parle\RParser::nonassoc — Declare a token with no associativity

Description

public void Parle\RParser::nonassoc ( string $tok )

Declare a terminal, that cannot appear more than once in the row.

Parameters

tok

Token name.

Return Values

No value is returned.
